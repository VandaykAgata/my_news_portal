# News Portal — Django Project

Учебный проект новостного портала, выполненный в рамках курса Python-разработчик.

## Технологические изменения и база данных

* **PostgreSQL**: Проект полностью переведен с дефолтной БД SQLite3 на профессиональную СУБД **PostgreSQL**. Это обеспечивает надежное хранение данных и поддержку сложных запросов.
* **Celery & Redis**: Реализована система фоновых задач и еженедельных рассылок новостей.

## Система логирования (Задание 4.1)

В проекте настроена комплексная система логирования по следующим правилам:

1. **Консоль (DEBUG режим)**: 
    - Сообщения `DEBUG` и выше.
    - Для `WARNING` и выше — выводится путь к файлу (`pathname`).
    - Для `ERROR` и `CRITICAL` — выводится полный стек ошибки (`exc_info`).
2. **Файлы логов**:
    - `general.log`: Общая информация (`INFO+`) из логгера `django`.
    - `errors.log`: Ошибки (`ERROR+`) из логгеров `request`, `server`, `template`, `db.backends`.
    - `security.log`: Логи безопасности (`django.security`).
3. **Email-уведомления**:
    - При `DEBUG=False` администраторы получают отчеты об ошибках уровней `ERROR` и выше.
